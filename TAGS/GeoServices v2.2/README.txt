GeoServices - MicroFramework

Contenido
---------

El paquete se encuentra distribuído de la siguiente forma:
	- lib: Contiene las librerías que conforman al núcleo del framework y deben ser
	       importados como referencia al proyecto
	
	- config: Contiene los xml que deben ser importados al directorio base,
		  los mismos contienen contenido demostrativo y auto-explicado
	
	- Ejemplos: Contiene 2 ejemplos (Tanto en Visual Basic .Net, como en C# .Net)
		    con aplicaciones simples realizadas con el Framework
		    La finalidad de ambas es mostrar el nombre de todos los Feature Classes
		    contenidos en una instancia de ArcSDE
		    Los archivos de configuración se encuentran completos, pero para ejecutar
		    las aplicaciones debe especificar una conexión SDE válida
		    
		· EjemploGeoService: Servicio de ejemplo utilizando las plantillas que provee
				     el Framework
		· EjemploGeoApp: Aplicación de ejemplo utilizando al Framework Out-of-the-box
	
	- Utils: Contiene la herramienta de encriptado GeoServices	

	- GeoServices - Manual del Desarrollador.doc: 
			Manual para el desarrollador de GeoServices
			Contiene toda la información necesaria para realizar una aplicación
			haciendo uso del Framework
